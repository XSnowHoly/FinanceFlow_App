- [x] 账单详情页

# 配置

根目录创建 `.env` 文件，写入以下内容

```
# 接口地址
VITE_API_URL=https://finance.xxx.com
# 本地测试代理地址
VITE_PROXY_API_URL=http://finance.xxx.com
```
